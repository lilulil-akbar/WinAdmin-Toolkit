# WinAdmin-Toolkit

Koleksi skrip automasi dan solusi teknis untuk administrasi sistem Windows. Repositori ini berisi berbagai alat bantu (utility) yang dikembangkan untuk menangani masalah IT operasional secara efisien.

---

## Deskripsi
Proyek ini adalah kumpulan solusi berbasis skrip (PowerShell & Batch) untuk memperbaiki masalah konfigurasi, optimasi sistem, dan automasi tugas-tugas IT Support. Fokus utama dari toolkit ini adalah efektivitas waktu dan standarisasi perbaikan sistem.

## Cara Penggunaan Umum
1. Hubungkan atau *Download* repositori ini ke mesin lokal.
2. Cari skrip yang dibutuhkan di dalam folder `Scripts`.
3. Sebelum menjalankan skrip, sangat disarankan untuk membaca file **README.md** terlebih dahulu didalam folder skrip yang dibutuhkan.
